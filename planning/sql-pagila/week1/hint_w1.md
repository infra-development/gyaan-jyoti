## Helpful Questions

For every problem ask:

1. Which table contains the data?
2. Do I need another table?
3. Do I need a JOIN?
4. Do I need GROUP BY?
5. Do I need an aggregate function?
6. Do I need sorting?
7. Do I need filtering?
